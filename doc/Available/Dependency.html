Overview

Represents a dependency (required or optional) of your system for which you can query your Available instance.

Direct Known Subclasses

def initialize(name, require_path=nil)
  @name         = name.to_sym
  @require_path = require_path || name.to_s
  @satisfied    = false
  @error        = nil
end

def require!
  yield if block_given?
  require @require_path
rescue LoadError => exception
  @error      = exception
  @satisfied  = false
else
  @error      = nil
  @satisfied  = true
end
